# AcademiCore – Taller N°4 POO

Sistema de gestión de certificaciones para estudiantes de Ingeniería, desarrollado en Java como parte del Taller N°4 de Programación Orientada a Objetos.

- **Axel Alcayaga** - Ingeniería Civil en Computación e Informática - 20.832.945-6

## 1. Descripción general

**AcademiCore** es una aplicación que permite administrar:

- Usuarios del sistema:
  - Administradores  
  - Coordinadores  
  - Estudiantes  
- Certificaciones (Desarrollo de Software, Sistemas Inteligentes, Ciberseguridad, etc.)
- Cursos asociados a cada certificación
- Registros de inscripción de estudiantes en certificaciones
- Notas de los estudiantes y progreso dentro de las certificaciones

El sistema incorpora **patrones de diseño** (Singleton, Factory, Strategy y Visitor) para cumplir con los requisitos del Taller y favorecer un diseño flexible y extensible.

## 2. Tecnologías y requisitos

- **Lenguaje:** Java 8+  
- **IDE recomendado:** Eclipse  
- **Tipo de proyecto:** Java Project
